# TypeScript Task 0 - Student Interface

This project demonstrates basic TypeScript concepts including:
- Interface declaration
- TypeScript variables
- DOM manipulation with TypeScript

## Requirements
- TypeScript
- Webpack
- ESLint

## Project Structure
- `js/main.ts` - Main TypeScript file with Student interface
- Configuration files for TypeScript, Webpack, and ESLint

## Usage
1. Install dependencies: `npm install`
2. Build project: `npm run build`
3. Run dev server: `npm run dev`
